## 快捷键
modkey = Mod4  
- 快速入门
mod+Shift+q 关闭当前窗口  
mod+Shift+r 重启Spectwm  
mod+j       切换上一个窗口  
mod+k       切换下一个窗口  
mod+Shift+j  
mod+Shift+k 移动窗口到上一窗口  
mod+Shift+j 移动窗口到下一个窗口  
mod+e       打开文件管理器  
- 更多快捷键查看配置文件  
  
## 安装
```shell
	git clone https://github.com/denstiny/spectrwm.conf.git ~/.config/spectrwm
	reboot
	
```
